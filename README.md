# Mono-Repo with GameTable backend and frontend

## Frontend
Bun + React

## Backend
Bun + Ts-Node + Express + Mysql + Docker


To install dependencies:

```bash
bun install
```

To run:

```bash
bun run start
```

To debug:
```bash
bun --watch index.ts
```

Check server uptime status and database connection:
```bash
curl http://localhost:8000/health
```
